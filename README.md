# 🎮 Sync Survivor

A survival arcade game inspired by real-world IT system chaos.

## 🚀 Features
- Canvas game engine
- Mobile + desktop support
- Touch controls
- Dynamic difficulty
- Coffee slow-motion mode ☕
- Clean UI + HUD

## 🎯 Goal
Survive system sync chaos until Friday 5:00 PM.

## 🎮 Controls
Desktop:
- Arrow keys / A-D

Mobile:
- Drag left/right

## 🛠 Tech
- HTML5 Canvas
- JavaScript
- CSS

## 🚀 Run
Open index.html
